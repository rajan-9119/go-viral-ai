# Go Viral AI

AI-powered content virality analyzer.

## Features
- Caption analysis
- Image & video upload
- Virality score (0–100)
- Hook, Emotion, CTA breakdown
- Suggestions & hashtags
- Improved caption
- Competitor insights

## Tech Stack
- Django (Backend)
- Groq API (AI)
- HTML, CSS, JS (Frontend)

##  How to Run
1. Clone repo
2. Install requirements:
   pip install -r requirements.txt
3. Add .env file with GROQ_API_KEY
4. Run:
   python manage.py runserver
